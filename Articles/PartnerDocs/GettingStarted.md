## <a name="1-installation-and-quick-start"></a>1. Installations- und Schnellstart 
  * Kaizala aus [Google wiedergeben Speicher](https://play.google.com/store/apps/details?id=com.microsoft.mobile.polymer&hl=en) für Android oder [App Store](https://itunes.apple.com/in/app/microsoft-kaizala/id1112208399?mt=8) für iOS jetzt herunterladen
  * Aktivieren Sie mit Ihrer Telefonnummer und loslegen
  * Klicken Sie auf das Zeichen '+' zum Erstellen einer Gruppe und Hinzufügen von Mitgliedern aus der Kontaktliste oder Rufnummern direkt hinzufügen
  * Benennen Sie der Gruppe, und Sie festgelegt sind
  * Senden Sie eine Aktion Kaizala, indem Sie auf der oberen rechten Ecke, um die Palette Kaizala Aktionen aufzurufen
  * Versuchen Sie schnell eine Umfrage oder eine Ankündigung finden in der Praxis Kaizala senden. 
  Wenn Sie alle Benutzer in Ihrem Netzwerk auf Kaizala besitzen, können Sie zum der Gruppe hinzufügen, und klicken Sie dann auf den Gruppennamen, um eine Einladung an alle Mitglieder senden, die nicht in Kaizala sind.

### <a name="31-create-groups-and-add-users"></a>3.1 Gruppen erstellen und Hinzufügen von Benutzern 

  Nachdem Sie die Gruppentypen und die Struktur für Ihre Organisation identifiziert haben, können Sie entweder direkt mit der app Gruppen erstellen oder Sie können die Kaizala-Verwaltungsportal Gruppen erstellen und Hinzufügen von Benutzern in einer Sammeloperation mit Excel. <br/><br/>
  Sie können eine Gruppe mit dem das Zeichen '+' erstellen und Hinzufügen von Mitgliedern dieser Gruppe. Wenn Sie untergeordneten Gruppen hinzufügen möchten, können Sie tun, dass auf dem gleichen Bildschirm, auf dem Sie sind, Mitglieder hinzufügen – wählen Sie einfach Gruppen anstelle von Kontakten und alle vorhandene Gruppen auf dem Telefon angezeigt werden. Sie müssen die einzelne untergeordneten zuerst Gruppen erstellen, und klicken Sie dann die übergeordnete Gruppe hinzugefügt werden. <br/><br/>
  
  ** Hier erhalten Sie schnell eine Anleitung zum Erstellen dieser Gruppen mithilfe des Portals: ** 
 <ol>
<li>Mit Ihrem Office 365 oder Kaizala Pro Anmeldeinformationen-Anmeldung auf [Kaizala-Verwaltungsportal] [5]</li>
<li>    Klicken Sie auf Hochladen der CSV-Datei unter Hinzufügen von Benutzern in einer Sammeloperation</li>
<li>    Herunterladen von Excel-Vorlagen und Mobiltelefonnummern und Hierarchieinformationen, hinzufügen, falls vorhanden</li>  
<li>    Durchsuchen Sie, und fügen Sie die beiden Dateien, und klicken Sie auf Submit</li>
<li>    Nachdem Sie die Informationen auf dem Bildschirm anzuzeigen, klicken Sie auf Hochladen</li>
<li>    Wenn Sie öffentliche Gruppen erstellen möchten, klicken Sie im linken Menü auf Öffentliche Gruppe und durchlaufen Sie Flow erstellen</li>
  </ol>
    Ausführliche Informationen zum Erstellen von Gruppen und Hinzufügen von Mitgliedern finden Sie [here.] [6]

> Kaizala kann aus [Google wiedergeben Speicher](https://play.google.com/store/apps/details?id=com.microsoft.mobile.polymer&hl=en) für Android oder [App Store](https://itunes.apple.com/in/app/microsoft-kaizala/id1112208399?mt=8) für iPhones installiert werden.

### <a name="41-get-kaizala-installed-by-your-organizational-users"></a>4.1 abrufen Sie Kaizala durch Ihre Organisation Benutzer installiert

App-Installation in Ihrer Org gesteuerter wäre sehr wichtig für den Erfolg dieses Projekts. Es folgen einige Methoden, die von anderen Kunden:
* Klicken Sie auf den Gruppennamen im Kaizala app aus, und klicken Sie auf alle einladen, um eine Textnachricht an alle Mitglieder einladen zu senden.
* Senden einer e-Mails an Personen freigeben Wert-Eigenschaft und einen Link zum Herunterladen.
* Versenden Sie Nachrichten an den Organisationseinheiten WhatsApp Gruppen mit den Link herunterladen.   
* Versenden Sie Bulk SMSs für Personen mit den Link herunterladen. 
* Entwerfen von Kaizala Poster, und platzieren sie an den Stellen, wobei jeder wird angezeigt.
* Incentivize Personen zum Installieren von Kaizala mit Early Bird Gewinner oder mit Quizfragen oder bewährte Bild Übermittlung Umfrage usw.

### <a name="42-find-users-who-have-installed-kaizala--user-activation-data"></a>4.2, suchen Sie nach Benutzern, die Kaizala installiert haben / Aktivierung von Benutzerdaten

    From the Kaizala Management Portal, you can download a CSV file that shows the status of Kaizala for each user.
<ol>
<li>Mit Ihrem Office 365 oder Kaizala Pro Anmeldeinformationen-Anmeldung auf [Kaizala-Verwaltungsportal] [5]</li>  
<li>Klicken Sie auf der Registerkarte Gruppen auf der linken Seite, und klicken Sie auf den Gruppennamen</li>
<li>Sie können sehen, die Mitglieder registriert oder nicht registriert (Kaizala nicht installiert). Sie können auch die Benutzer exportieren > neue Anforderung initiieren > Option zum Abrufen dieser Informationen in einer CSV-Datei herunterladen</li> </ol>

### <a name="43-find-whether-users-are-using-kaizala--kaizala-usage-data"></a>4.3 suchen, ob Benutzer Kaizala verwenden / Kaizala-Verwendungsdaten

    You can monitor whether users are using Kaizala using the Kaizala Management Portal.
<ol>

<li>Klicken Sie auf der Registerkarte Gruppen auf der linken Seite, und klicken Sie auf den Gruppennamen</li>
<li>Sie können klicken Sie auf Berichte im linken Bereich auf und klicken Sie auf Verwendung Telemetrie zum Überwachen der Verwendung von Kaizala in Ihren Gruppen</li> </ol>
