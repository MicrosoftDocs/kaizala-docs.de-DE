## <a name="1-installation-and-quick-start"></a>1. Installation und Schnellstart 
  * Herunterladen von Kaizala aus dem [Google Play Store](https://play.google.com/store/apps/details?id=com.microsoft.mobile.polymer&hl=en) für Android oder [App Store](https://itunes.apple.com/in/app/microsoft-kaizala/id1112208399?mt=8) für IOS now
  * Aktivieren Sie Ihre Telefonnummer und beginnen Sie sofort
  * Klicken Sie auf das "+"-Zeichen, um eine Gruppe zu erstellen und Mitglieder aus Ihrer Kontaktliste hinzuzufügen oder die Telefonnummern direkt hinzuzufügen.
  * Geben Sie der Gruppe einen Namen, und Sie sind alle festgelegt.
  * Versuchen Sie, eine Kaizala-Aktion zu senden, indem Sie auf die obere rechte Ecke klicken, um die Kaizala-Aktions Palette aufzurufen.
  * Senden Sie eine Schnellabfrage oder eine Ansage, um Kaizala in Aktion zu sehen. 
  Wenn Sie niemanden in Ihrem Netzwerk auf Kaizala haben, können Sie diese weiterhin der Gruppe hinzufügen und dann auf Gruppen Name klicken, um eine Einladung an alle Mitglieder zu senden, die sich nicht in Kaizala befinden.

## <a name="3define-groups-in-kaizala-for-your-org"></a>3. Definieren von Gruppen in Kaizala für Ihre Organisation  

Nachdem Sie die Anwendungsfälle für Kaizala in Ihrer Organisation ermittelt haben, müssen Sie die richtigen Gruppen in Kaizala erstellen, damit diese Anwendungsfälle effektiv erreicht werden können. <br/><br/>
Kaizala bietet drei Arten von Gruppen, die isoliert oder in Verbindung verwendet werden können, um die verschiedenen Kommunikations-und Zusammenarbeitsanforderungen Ihrer Organisation zu unterstützen – sowohl innerhalb seiner Mitarbeiter als auch mit der Wertschöpfungskette oder den Mitarbeitern.  <br/><br/>

### <a name="31-create-groups-and-add-users"></a>3,1 Gruppen erstellen und Benutzer hinzufügen 

  Nachdem Sie die Gruppentypen und die Struktur für Ihre Organisation identifiziert haben, können Sie entweder Gruppen direkt über die APP erstellen, oder Sie können mithilfe des Kaizala-Verwaltungsportals Gruppen erstellen und Benutzer mithilfe von Excel massenweise hinzufügen. <br/><br/>
  Sie können eine Gruppe mit dem Zeichen "+" erstellen und dieser Gruppe Mitglieder hinzufügen. Wenn Sie untergeordnete Gruppen hinzufügen möchten, können Sie dies auf demselben Bildschirm tun, auf dem Sie Mitglieder hinzufügen – wählen Sie einfach Gruppen anstelle von Kontakten aus, und alle vorhandenen Gruppen auf Ihrem Telefon werden angezeigt. Sie müssen zuerst die einzelnen untergeordneten Gruppen erstellen und diese dann der übergeordneten Gruppe hinzufügen. <br/><br/>
  
  * * Hier finden Sie eine kurze Anleitung zum Erstellen dieser Gruppen mithilfe des Portals: * * 
 <ol>
<li>Anmelden [Kaizala Management Portal] [5] mit Ihren Office 365-Anmeldeinformationen oder Kaizala pro-Anmeldeinformationen</li>
<li>    Klicken Sie unter Add users in Bulk auf Upload CSV file.</li>
<li>    Herunterladen von Excel-Vorlagen und hinzufügen mobiler Nummern und Hierarchieinformationen, falls vorhanden</li>  
<li>    Durchsuchen und Hinzufügen der Dateien und klicken auf Senden</li>
<li>    Sobald die Informationen auf dem Bildschirm angezeigt werden, klicken Sie auf hochladen.</li>
<li>    Wenn Sie öffentliche Gruppen erstellen möchten, klicken Sie im linken Menü auf öffentliche Gruppe und dann auf erstellen.</li>
  </ol>

### <a name="41-get-kaizala-installed-by-your-organizational-users"></a>4,1 Get Kaizala von Ihren organisatorischen Benutzern installiert

Das Steuern der App-Installation in ihrer org wäre für den Erfolg dieses Projekts sehr wichtig. Hier finden Sie einige Methoden, die von anderen Kunden verwendet werden:
* Klicken Sie in der Kaizala-App auf Gruppenname, und klicken Sie auf alle einladen, um eine Textnachricht zu senden, um alle Mitglieder einzuladen.
* Senden Sie eine e-Mail an People Sharing Value Prop und einen Link zum herunterladen.
* Senden von Nachrichten an die Organisations-WhatsApp-Gruppen mit dem Download-Link.   
* Senden Sie Bulk-SMS an Personen mit dem Download-Link. 
* Entwerfen Sie Kaizala Poster, und platzieren Sie Sie an Stellen, an denen Sie von jedem angezeigt werden können.
* Anreize Personen zur Installation von Kaizala mit Early Bird Winner oder mit Quiz oder Best Picture Submission Survey usw.

### <a name="42-find-users-who-have-installed-kaizala--user-activation-data"></a>4,2 Benutzer mit Kaizala/Benutzer Aktivierungsdaten suchen

    From the Kaizala Management Portal, you can download a CSV file that shows the status of Kaizala for each user.
<ol>
<li>Anmelden [Kaizala Management Portal] [5] mit Ihren Office 365-Anmeldeinformationen oder Kaizala pro-Anmeldeinformationen</li>  
<li>Klicken Sie Links auf die Registerkarte Gruppen, und klicken Sie auf den Namen Ihrer Gruppe.</li>
<li>Sie können sehen, dass die Mitglieder registriert oder nicht registriert sind (Kaizala nicht installiert). Sie können auch die Option "Benutzer > Initiieren einer neuen Anforderung >" verwenden, um diese Details in einer CSV-Datei abzurufen.</li> </ol>

### <a name="43-find-whether-users-are-using-kaizala--kaizala-usage-data"></a>4,3 ermitteln, ob Benutzer Kaizala/Kaizala-Verwendungsdaten verwenden

    You can monitor whether users are using Kaizala using the Kaizala Management Portal.
<ol>
<li>Anmeldung im [Kaizala-Verwaltungs Portal](http://manage.kaiza.la/) mit ihren Office 365-Anmeldeinformationen oder Kaizala pro-Anmeldeinformationen</li>  
<li>Klicken Sie Links auf die Registerkarte Gruppen, und klicken Sie auf den Namen Ihrer Gruppe.</li>
<li>Klicken Sie im linken Bereich auf Berichte, und klicken Sie auf Usage teleMetrie, um die Verwendung von Kaizala in ihren Gruppen zu überwachen.</li> </ol>
